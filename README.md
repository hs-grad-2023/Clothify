# 가상 피팅 & 나만의 옷장 서비스
- 본 서비스는 AI와 컴퓨터 비전 기술을 중심으로 한 혁신적인 옷장 관리 웹사이트를 제시
- 장고를 기반으로 한 웹사이트는 가상 피팅 기능을 핵심으로 다양한 기능을 제공하여 사용자들이 온라인상에서도 옷을 효과적으로 관리하고 코디할 수 있게 한다.
- 최종 버전은 branch master
